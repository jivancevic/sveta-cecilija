# Google Business Profile weekly post template

How to use: open `business.google.com` → HGD Sveta Cecilija → **Promote** → **Add update** (or **Event** for upcoming shows). Use the Event type when there's a concrete date/time, it shows richer in search results. Use the Update type for general announcements.

Post each Monday for the week's shows. One post per show is fine; or one combined "this week" post if there are multiple.

---

## Template A: Event post (preferred, one per show)

**Event title**
- EN: `Moreška sword dance, [date]`
- HR: `Moreška, [datum]`

**Start / end**
Set the actual show time (e.g. `Thu 11 June 2026, 21:00 to 22:00`).

**Cover photo**
A high-quality performance shot. Reuse the same one across the season for visual consistency, unless you have a new standout image.

**Description (EN)**
```
Korčula's 500-year-old sword dance, performed by HGD Sveta Cecilija, Croatia's oldest active moreška ensemble (since 1883).

📅 [Day, Date]
🕘 [Time]
📍 [Venue: Summer Cinema or Cultural Center Korčula]
🎟️ Tickets: moreska.eu/tickets

Adult €20 · Child €10
```

**Description (HR)**
```
Korčulanska viteška igra mačevima, izvedba HGD Sveta Cecilija, najstarijeg aktivnog moreškarskog ansambla u Hrvatskoj (od 1883.).

📅 [Dan, datum]
🕘 [Vrijeme]
📍 [Mjesto: Ljetno kino ili Centar za kulturu]
🎟️ Ulaznice: moreska.eu/tickets

Odrasli €20 · Djeca €10
```

**Button**
"Book" → `https://moreska.eu/tickets`

## Template B: Weather-fallback notice

Use when forecast looks bad and the venue is moving to Centar za kulturu. Post 24h before the show.

**EN**
```
Tonight's moreška moves indoors to the Cultural Center Korčula due to weather. Same time, same tickets, same dance, just covered. See you there.

🕘 [Time]
📍 Cultural Center Korčula (Centar za kulturu)
🎟️ moreska.eu/tickets
```

**HR**
```
Večerašnja moreška seli u Centar za kulturu zbog vremenskih uvjeta. Isto vrijeme, iste ulaznice, ista igra, samo pod krovom. Vidimo se.

🕘 [Vrijeme]
📍 Centar za kulturu
🎟️ moreska.eu/tickets
```

## Cadence checklist

- **Monday morning**: post Event entries for every show that week.
- **Day-of, if weather pivots**: post Template B by 14:00.
- **End of season (October)**: thank-you post with attendance figure.
- **Late May**: Template C to open the new season.

## Style rules

- Never use em-dashes (—) in user-facing copy. Use commas, colons, or periods.
- Croatian "moreška" is lowercase except at sentence start. English "Moreška" is always capitalised.
- Always link `moreska.eu/tickets`, never the legacy `korcula-moreska.com`.
- Venue names follow CLAUDE.md: EN "Summer Cinema" / "Cultural Center Korčula"; HR "Ljetno kino" / "Centar za kulturu".
